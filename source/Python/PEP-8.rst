


Nameing Style
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Function  function/my_function
Variable  x/var/my_variable
Class     Model/MyClass
Method    class_method/Method
Constant  CONSTANT/MY_CONSTANT
Module    mudule.py/my_mudule.py
package   package/mypackage


How to Choose Names
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
尽量一直使用最简洁且最具描述性的名称

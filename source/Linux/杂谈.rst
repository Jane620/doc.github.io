vim批量操作
~~~~~~~~~~~~~~~~~~

::
  批量替换
  :%s#findwords#replacewords#g

  批量删除
  在view视图下
  X dd   (X为当前鼠标焦点所在行向下X的全部内容)



拷贝插入
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
非编辑状态下yy，拷贝当前行，在当前行输入p，自动粘贴到下一行


带上密码方式执行linux远程机器命令
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
sshpass -p passwd ssh username@ip "echo xxx"


linux交互中输入通过脚本预设
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
<<EOF
content
EOF
eg. passwd需要输入两次密码
passwd sct <<EOF
111
111
EOF
